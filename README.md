# Pharmacy-Sales-Analytics
Power BI project analyzing pharmacy sales performance using interactive dashboards and KPIs.

## Project Overview
Business Problem

## Objectives
Analyze pharmacy sales performance using Power BI.
Track overall sales, quantity sold, and transaction trends.
Identify the top-performing doctors based on sales.
Compare sales performance across different doctors.
Monitor monthly sales trends to identify growth patterns.
Build an interactive dashboard using filters and slicers for easy analysis.
Generate actionable insights to support data-driven business decisions.

## KPIs
💰 Total Sales
📦 Total Quantity Sold
🧾 Total Transactions
💵 Average Sale per Transaction
👨‍⚕️ Total Doctors
🏆 Top Performing Doctor
📊 Average Sales per Doctor

## Tools Used
Power BI – Data visualization and interactive dashboard creation.
Microsoft Excel – Data storage, cleaning, and preparation.
Power Query – Data transformation and preprocessing.
DAX (Data Analysis Expressions) – Calculated measures, KPIs, and business

## Key Insights
Identified the top-performing doctors based on total sales.
Compared sales performance across different doctors to highlight high and low contributors.
Analyzed monthly sales trends to identify periods of higher and lower sales.
Evaluated total quantity sold and total transactions to understand sales volume.
Calculated the average sales per transaction to measure transaction value.
Used interactive slicers to analyze sales by doctor, product, and time period.
Developed an interactive Power BI dashboard to support faster, data-driven decision-making.
